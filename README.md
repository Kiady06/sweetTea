# 🍵 Sweet Tea - Salon de Thé Artisanal

Sweet Tea est un projet de site web vitrine pour Sweet tea By family. L'objectif est de présenter une sélection de thés de haute qualité dans une ambiance douce et chaleureuse.

## 🚀 Aperçu
Le site permet aux utilisateurs de :
* Consulter la carte des thés (Verts, Noirs, Infusions).
* Découvrir l'histoire de la boutique.
* Localiser le salon de thé et voir les horaires d'ouverture.
* Prendre contact via un formulaire dédié.

## 🛠️ Technologies utilisées
* **HTML5** : Structure sémantique du contenu.
* **CSS3** : Design responsive (Flexbox/Grid) et animations douces.
* **Google Fonts** : Utilisation de polices élégantes.
